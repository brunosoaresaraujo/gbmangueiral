Como funciona a renderização?
Utilizamos um pipeline WebGL personalizado misturado com elementos CSS DOM padrão para obter fundos ambientais de alta taxa de quadros sem sacrificar a acessibilidade.
É totalmente responsivo?
Sim, cada componente é construído com tipografia fluida e CSS grid/flexbox para se adaptar perfeitamente a dispositivos móveis e tablets.
